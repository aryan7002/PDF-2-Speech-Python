# PDF-2-Speech
>## Features
- this prject will read the content of PDF for you

- it also shows the text of pdf in terminal wich it is going to read

- it will also display the number of pages in pdf in the terminal

>## package to install
- 1 pyttsx3  
  - `pip install pyttsx3`

- 2 PyPDF2
  - `pip install PyPDF2`

>## Things to change
- name of the pdf
- pdf
